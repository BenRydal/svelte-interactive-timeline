# Changelog

## [0.1.0](https://github.com/BenRydal/svelte-interactive-timeline/releases/tag/v0.1.0) (2026-02-04)

### Features

* Initial release of svelte-interactive-timeline
* Core types and utilities for timeline management
* Svelte 5 runes-based reactive store (`createTimelineStore`)
* Factory functions for customizable colors and layout
* Canvas-based rendering with injectable layer system
* Built-in layers: Background, Playhead, Hover, ZoomSelection
* Timeline, TimelineCanvas, and TimelineControls components
* Full TypeScript support
* DaisyUI/Tailwind compatible styling
* Demo page with usage examples
